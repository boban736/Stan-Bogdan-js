## Ответы на контрольные вопросы

1. Какие методы массивов можно использовать для обработки объектов в JavaScript?

Чаще всего используются `map`, `filter`, `find`, `reduce`, `forEach`, `some` и `every`.

`filter` удобно использовать для отбора объектов по условию. `find` возвращает первый найденный объект. `map` нужен, когда из массива объектов нужно получить массив других значений, например только описания транзакций. `reduce` подходит для подсчета суммы или создания объекта-счетчика.

2. Как сравнивать даты в строковом формате в JavaScript?

Если дата записана как строка, ее можно преобразовать в объект `Date`:

```js
const date = new Date("2023-10-01");
```

После этого даты можно сравнивать через `getTime()` или обычными операторами сравнения. В моей работе я использовал преобразование через `new Date(...)`, потому что так проще сравнивать диапазоны.

3. В чем разница между `map()`, `filter()` и `reduce()`?

`map()` возвращает новый массив такой же длины, но с преобразованными значениями.

`filter()` возвращает новый массив только с теми элементами, которые прошли проверку.

`reduce()` возвращает один итоговый результат: число, строку, объект или другое значение.
